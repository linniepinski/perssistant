== Description ==

This plugin is an addon plugin which will work only with wordpress contact form 7
this plugins uses the API services from <a href="https://www.cognalys.com" alt="mobile number verification for free">www.cognalys.com</a>. All registered users in <a href="https://www.cognalys.com" alt="mobile number verification for free">www.cognalys.com</a>is powered with 500 free mobile number verifications per day.

This plugin supports the webmasters to verify the mobile number of the contact form users by a missed call verificaton OTP / two factor authentication.
